### Comando para preparar archivos a guardar en el repositorio
    $ git add . 
    $ git add -A
    $ git add *home*
    $ git add index.html

### Comando para deshacer la preparacion de los archivos a guardar

    $ git rm --cached <file>

### Comando para guardar en el repositorio local

    $ git commit -m "Este es una prueba"
    $ git commit
